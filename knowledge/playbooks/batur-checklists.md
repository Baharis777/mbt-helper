# Чеклисты по Батуру

## Перед рекомендацией варианта

- [ ] Дата поездки.
- [ ] Район отеля.
- [ ] Количество людей.
- [ ] Есть ли дети.
- [ ] Возраст детей.
- [ ] Есть ли ограничения по здоровью.
- [ ] Есть ли проблемы с коленями, дыханием, сердцем, беременность или слабая физическая форма.
- [ ] Есть ли собственный транспорт.
- [ ] Нужен ли pickup из отеля.
- [ ] Бюджет или чувствительность к цене.
- [ ] Нужны ли горячие источники.
- [ ] Нужен ли групповой или приватный формат.
- [ ] Нужен ли формат без пешего восхождения.
- [ ] Есть ли жесткое ограничение по времени возвращения.

## Перед подтверждением бронирования

- [ ] Доступность проверена у организатора.
- [ ] Live-статус Батура проверен, если есть вопрос про безопасность или ближайшую дату.
- [ ] Конкретный вариант выбран и подтвержден.
- [ ] Цена подтверждена.
- [ ] Валюта подтверждена.
- [ ] Что включено подтверждено.
- [ ] Что не включено подтверждено.
- [ ] Pickup или место встречи подтверждены.
- [ ] Район pickup подходит клиенту.
- [ ] Время выезда подтверждено.
- [ ] Длительность или ожидаемое возвращение подтверждены, если важно.
- [ ] Правила оплаты объяснены.
- [ ] Правила отмены или переноса объяснены.
- [ ] Условия при плохой погоде объяснены, если они подтверждены.

## Перед ответом на вопрос о безопасности

- [ ] Проверен `knowledge/live-status/batur.md`.
- [ ] Проверен организатор или утвержденный источник.
- [ ] Зафиксировано время проверки.
- [ ] Зафиксировано, кто подтвердил статус.
- [ ] Не используются слова "безопасно", "открыто", "доступно", "тур состоится" без подтверждения.

## Перед рекомендацией для семьи

- [ ] Возраст детей уточнен.
- [ ] Возрастные правила варианта проверены.
- [ ] Нагрузка проверена.
- [ ] Формат трансфера проверен.
- [ ] Live-статус проверен при необходимости.
- [ ] Менеджер не обещает участие ребенка без подтверждения.

## Перед рекомендацией бюджетного варианта

- [ ] Клиент подтвердил бюджетный приоритет.
- [ ] Проверено, есть ли собственный транспорт.
- [ ] Проверено, нужен ли pickup.
- [ ] Сравнена итоговая стоимость подходящих вариантов.
- [ ] Самый дешевый вариант не конфликтует со здоровьем, логистикой и целью клиента.

## Перед рекомендацией варианта с горячими источниками

- [ ] Клиент подтвердил, что горячие источники важны.
- [ ] `hot_springs_included` подтверждено в матрице или у организатора.
- [ ] Проверено, входят ли билеты или только остановка.
- [ ] Проверена длительность программы.
- [ ] Цена и включения подтверждены.
